# Description
This is a simple AI based chatbot that uses both the Gemini API to give wine pairings based on food. 

## Dependencies

### .env and API key
To run, a .env file must be created in the root directory of the repo and then a Gemini API key must be added as an environment variable:
* `GEMINI_API_KEY=insert api key here`.

### required libraries
* Must use composer to install the vlucas/phpdotenv library. Use command: `composer require vlucas/phpdotenv` in terminal when inside the repository root directory.
